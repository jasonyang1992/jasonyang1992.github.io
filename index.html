Recent Projects

Web Programming Academy

Web Programming Academy is a website for anyone who wants to explore web programming. It is a tutorial website for beginner or someone who is interest in making their own website. The tutorial website contains detail explaination for HTML, CSS, and JavaScript code.

Question Game

Question Game is a game that will think of what you are thinking. To do so, it will ask you six questions to determine what you are thinking. After that, on the seventh question, the program will then ask what you are thinking. If the program is sucessful in guessing what you are thinking, the program wins. If the program guesses wrong, you win.

College Cooking

College Cooking is a website for College Students that want to learn how to cook simple recipes or create a daily meal for themselves during their college years. This is a project that I worked on as a class project. I used a bootstrap layout to map the website. The websites starts as a random hierarchy and then laters goes into a linear hierarchy.
